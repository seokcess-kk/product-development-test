# Wireframe: Calendar

## Overview

- **화면**: 캘린더 (일정 관리)
- **경로**: `/calendar`
- **목적**: 주간/일간 학습 일정 확인 및 관리
- **진입점**: 대시보드, 하단 네비게이션
- **이탈점**: `/dashboard`, `/records`, `/plan/new`

## States

### Schedule Block States

#### Default (미완료, 미래)
```
┌───────────────────────────────────┐
│  [수학]                           │  Subject Badge Color
│  미적분 개념                       │
│  [ ]                              │  Empty checkbox
└───────────────────────────────────┘
```
- 배경: 과목별 밝은 색상 (예: primary-100)
- 텍스트: 과목별 진한 색상 (예: primary-700)
- 체크박스: 빈 원

#### Current (현재 시간대)
```
┌───────────────────────────────────┐
│  [수학]                    [현재] │  "현재" Badge
│  문제 풀이                        │
│  [ ]   [시작하기]                 │  Action Button
└───────────────────────────────────┘
```
- 테두리: 과목 색상 2px
- "현재" 배지: primary-500
- "시작하기" 버튼 표시

#### Completed (완료)
```
┌───────────────────────────────────┐
│  [수학]                           │
│  ~~미적분 개념~~                   │  Strikethrough
│  [v]  55분                        │  Actual time
└───────────────────────────────────┘
```
- 배경: gray-100
- 텍스트: gray-400, 취소선
- 체크박스: 체크, success-500
- 실제 학습 시간 표시

#### Overdue (미완료, 지남)
```
┌───────────────────────────────────┐
│  [영어]                    [지남] │  "지남" Badge
│  문법 복습                        │
│  [ ]                              │
└───────────────────────────────────┘
```
- 테두리: warning-500
- "지남" 배지: warning-500

## Drag & Drop (P1 기능)

### Desktop - Drag to Move
```
┌───────────────────────────────────┐
│  [수학]          <-- Dragging     │  Elevated, shadow-lg
│  미적분 개념                       │  Opacity 0.8
│  [ ]                              │
└───────────────────────────────────┘

       │
       v  Drop Target (다른 시간대)

┌───────────────────────────────────┐
│      Drop here                    │  Dashed border
│      (놓으면 이동)                 │  primary-200
└───────────────────────────────────┘
```

### Mobile - Long Press to Move
1. 일정 길게 누르기 (500ms)
2. 햅틱 피드백 (지원 시)
3. 드래그 모드 진입
4. 다른 위치에 드롭
5. 확인 토스트: "일정이 이동되었어요"

## Responsive Behavior

| Breakpoint | 변화 |
|------------|------|
| < 640px | 주간 뷰: 가로 스크롤, 일간 뷰: 리스트 |
| >= 640px | 주간 뷰: 전체 표시, 일간 뷰: 넓은 카드 |
| >= 1024px | 주간 뷰 확장, 시간 열 표시, 드래그 최적화 |

## Annotations

1. **View Toggle**
   - 위치: 헤더 아래
   - Segmented Control 스타일
   - 선택된 뷰: primary 색상

2. **Week/Day Navigator**
   - 화살표 버튼: 이전/다음
   - 날짜 범위 또는 단일 날짜 표시
   - "오늘" 버튼: 오늘 날짜로 빠르게 이동

3. **Weekly Calendar**
   - 시간표 형식 (09:00 ~ 22:00)
   - 30분 단위 그리드
   - 과목별 색상 (design-system 참조)
   - 완료 일정: 그레이아웃

4. **Daily List**
   - 시간순 정렬
   - 현재 시간대 강조
   - 완료/미완료 구분
   - 빈 시간대는 표시하지 않음

5. **Schedule Block**
   - 과목 배지: 좌측 상단
   - 제목: 1줄, 말줄임 (...)
   - 체크박스: 우측 하단 또는 좌측
   - 드래그 가능 표시 (데스크톱)

6. **Modal**
   - 중앙 정렬 (데스크톱)
   - Bottom Sheet (모바일)
   - 닫기 버튼: 우측 상단
   - 주요 액션: 우측 하단
