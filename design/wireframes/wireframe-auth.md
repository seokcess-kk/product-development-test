# Wireframe: Authentication Screens

## Overview

- **화면**: 로그인 / 회원가입
- **경로**: `/login`, `/signup`
- **목적**: 사용자 인증 및 계정 생성
- **진입점**: 랜딩 페이지, 직접 URL, 보호된 페이지 리다이렉트
- **이탈점**: `/dashboard` (로그인 성공), `/onboarding/profile` (회원가입 성공)

## Elements

### Login Elements

| 요소 | 타입 | 설명 | 인터랙션 |
|------|------|------|----------|
| 뒤로가기 | Icon Button | 이전 페이지 | history.back() |
| Logo | Image | 서비스 로고 | 클릭 시 `/` 이동 |
| Google 버튼 | Button (Social) | Google OAuth | OAuth 팝업 실행 |
| 이메일 입력 | Input (email) | 이메일 주소 | 필수, 유효성 검증 |
| 비밀번호 입력 | Input (password) | 비밀번호 | 필수, 표시/숨김 토글 |
| 눈 아이콘 | Icon Button | 비밀번호 표시 | 토글 동작 |
| 비밀번호 찾기 | Link (text) | 비밀번호 재설정 | `/forgot-password` 이동 |
| 로그인 버튼 | Button (Primary) | 로그인 실행 | 폼 제출 |
| 회원가입 링크 | Link (text) | 회원가입 페이지 | `/signup` 이동 |

## States

### Login States

#### Default
- 모든 입력 필드 비어있음
- 로그인 버튼 활성화 상태

#### Filled
- 입력 완료 상태
- 유효성 검사 통과 시 버튼 유지

#### Loading
- 로그인 버튼에 스피너 표시
- 입력 필드 비활성화
- "로그인 중..." 텍스트 (선택)

#### Error
```
┌───────────────────────────────────┐
│  이메일                            │
│  ┌─────────────────────────────┐  │
│  │  wrong@email.com      [!]   │  │ Error border (danger-500)
│  └─────────────────────────────┘  │
│  [!] 등록되지 않은 이메일입니다     │ Error message (danger-500)
└───────────────────────────────────┘
```

#### Login Error (Toast/Banner)
```
┌───────────────────────────────────┐
│ [!] 이메일 또는 비밀번호가         │ Error Banner
│     올바르지 않습니다              │
└───────────────────────────────────┘
```

### Signup States

#### Default
- 모든 필드 비어있음
- 가입 버튼 비활성화 (또는 활성화 후 유효성 검사)

#### Validation In Progress
- 실시간 유효성 검사 표시
- 통과: 초록색 체크 아이콘
- 실패: 빨간색 X 아이콘 + 에러 메시지

#### Email Duplicate


#### Password Validation
```
┌───────────────────────────────────┐
```

```
┌───────────────────────────────────┐

│  ┌─────────────────────────────┐  │
```

#### Loading
- 가입 버튼에 스피너
- 모든 필드 비활성화

#### Success (회원가입 완료 -> 프로필 설정 이동)
- `/onboarding/profile`로 리다이렉트
- 이메일 가입 시 이메일 인증 페이지로 먼저 이동

## Responsive Behavior

| Breakpoint | 변화 |
|------------|------|
| < 640px | 전체 화면 폼, 세로 배치 |
| >= 640px | 카드 형태, 중앙 정렬 |
| >= 1024px | 회원가입 폼 일부 2열 배치 (비밀번호/확인, 이름/학년) |

## Annotations

1. **Logo**
   - 크기: 48px height
   - 상단 중앙 정렬 (모바일), 카드 상단 (데스크톱)

2. **Google 버튼**
   - Google 브랜드 가이드라인 준수
   - 아이콘 + 텍스트 조합
   - height: 44px

3. **입력 필드**
   - height: 44px (터치 친화적)
   - 라벨: 입력 필드 위
   - 에러: 입력 필드 아래, danger-500 색상

4. **비밀번호 토글**
   - 눈 아이콘 (보이기) / 눈금 아이콘 (숨기기)
   - 입력 필드 내부 우측

5. **학년 선택**
   - Segmented Control 스타일
   - 모바일: 2x2 그리드 또는 가로 스크롤
   - 데스크톱: 4개 가로 배치

6. **법적 고지**
   - text-xs (12px), gray-500
   - 이용약관, 개인정보처리방침 링크

- [x] 키보드 네비게이션 (Tab 순서)

## Navigation

### Login
- **뒤로가기**: 이전 페이지 (history.back)
- **Google 로그인 성공**: `/dashboard` (기존 사용자) 또는 `/onboarding/profile` (신규)
- **이메일 로그인 성공**: `/dashboard`
- **비밀번호 찾기 클릭**: `/forgot-password`
