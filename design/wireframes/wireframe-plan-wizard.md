# Wireframe: Plan Creation Wizard

## Overview

- **화면**: 계획 생성 위저드 (5단계)
- **경로**: `/plan/new`
- **목적**: 가이드형 질문을 통해 맞춤 학습 계획 자동 생성
- **진입점**: 대시보드 "새 계획 만들기", 하단 + 버튼, 온보딩 플로우
- **이탈점**: `/dashboard` (완료 또는 취소)
- **예상 소요 시간**: 3-5분

## Annotations

1. **Step Indicator**
   - 위치: 헤더 아래
   - 현재 단계 텍스트 + 진행률 바
   - 진행률 바: height 4px, rounded-full

2. **Option Cards**
   - 선택 시: border primary-500, 배경 primary-50
   - 아이콘: 48px, 중앙 정렬
   - 간격: gap-4

3. **Chip Group**
   - 선택된 칩: bg primary-500, text white
   - 미선택: bg white, border gray-300
   - 여러 줄 가능 (flex-wrap)

4. **Draggable List**
   - 드래그 핸들: 좌측 3줄 아이콘
   - 드래그 중: 그림자 추가, 배경 primary-50
   - 애니메이션: 부드러운 위치 변경

5. **Time Allocation**
   - Progress Bar: 과목별 색상 (design-system 참조)
   - +/- 버튼: 5% 단위 조절
   - 실시간 재계산 (합계 100% 유지)

6. **Schedule Preview (Step 5)**
   - 주간 캘린더 형태
   - 과목별 색상 블록
   - 탭 시 Bottom Sheet로 수정

7. **Completion Celebration**
   - Confetti 애니메이션 (가벼운 효과)
   - 계획 요약 카드
   - Primary CTA 강조
