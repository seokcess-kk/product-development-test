# Wireframe: Study Timer

## Overview

- **화면**: 학습 타이머
- **경로**: `/records`
- **목적**: 학습 시간 측정 및 기록
- **진입점**: 대시보드, 캘린더 일정, 하단 네비게이션
- **이탈점**: `/dashboard`, `/calendar`

## States

### Timer States

#### Idle (시작 전)
```
Timer Display: 00:00:00
Button: [시작]
Color: Gray
```
- 과목 선택 필수
- 과목 미선택 시 "시작" 버튼 비활성화

#### Running (진행 중)
```
Timer Display: 00:45:30 (counting up)
Buttons: [일시정지] [완료]
Color: Primary (animated pulse)
```
- 배경색 미세 변화 (optional)
- 화면 꺼짐 방지 (wakeLock)
- 탭 닫아도 localStorage로 유지

#### Paused (일시정지)
```
Timer Display: 00:45:30 (frozen)
Buttons: [계속하기] [완료]
Additional: [학습 취소]
Color: Gray, Blinking
```
- 일시정지 아이콘 표시
- "일시정지됨" 라벨

#### Completed (완료)
```
Completion Modal 표시
Summary + Memo 입력
```

### Pomodoro States

#### Focus Phase (집중)
```
Timer: 25:00 (countdown)
Label: "집중 시간"
Color: Primary
```

#### Break Phase (휴식)
```
Timer: 05:00 (countdown)
Label: "휴식 시간"
Color: Success (Green)
```

#### Long Break (긴 휴식)
```
Timer: 15:00 (countdown)
Label: "긴 휴식 시간"
Color: Secondary
```
- 4 포모도로 완료 후

### Validation States

#### Subject Not Selected
- "시작" 버튼 비활성화
- 힌트: "과목을 선택해주세요"

## Responsive Behavior

| Breakpoint | 변화 |

## Annotations

1. **Timer Display**
   - Font: Monospace 또는 Tabular Numbers
   - Size: text-6xl (모바일), text-7xl (데스크톱)
   - 진행 중: primary-500 색상
   - 일시정지: gray-400, 점멸 효과

2. **Subject Chips**
   - 선택된 칩: bg primary-500, text white
   - 미선택: bg white, border gray-300
   - 과목별 색상 적용 가능

3. **Buttons**
   - 시작: Primary, Large (height 56px)
   - 일시정지/완료: 나란히 배치, Medium
   - 학습 취소: Ghost, Danger 색상

4. **Pomodoro**
   - 토마토 아이콘
   - 원형 진행 표시 (Progress Ring)
   - 완료된 포모도로 카운트

5. **오늘 학습 요약**
   - 총 시간: text-3xl, font-bold
   - 과목별: 리스트, 과목 색상 배지

6. **기록 리스트**
   - 날짜별 그룹화
   - 날짜 헤더: sticky (스크롤 시 고정)
   - 항목: 과목 배지 + 시간 + 메모

7. **Completion Modal**
   - Bottom Sheet (모바일)
   - Center Modal (데스크톱)
   - 메모 입력: optional
   - 일정 연동: 자동 완료 처리
